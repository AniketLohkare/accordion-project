# Accordion Component

A simple accordion built using **only HTML and CSS**.  
This project is part of my frontend learning and UI practice.

## Demo
Live site: https://basic-accordion-component.netlify.app/

## Features
- Pure HTML and CSS (no JavaScript)
- Click to expand and collapse
- Smooth transition
- Minimal and clean UI

## Tech Stack
- HTML
- CSS
